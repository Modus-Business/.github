## 우리가 해결하는 문제

모둠 활동은 수업에서 자주 사용되지만, 실제 현장에서는 참여 불균형, 감정적인 표현, 기여도 판단의 어려움이 반복됩니다.

Modus는 AI가 학생을 평가하거나 대화를 통제하는 서비스가 아닙니다.  
학생들이 직접 대화하고 결정하는 흐름은 유지하면서, AI가 필요한 순간에만 표현을 완화하고 참여를 열어주며 논의를 더 깊게 만드는 협업 보조자입니다.

---

## Modus가 하는 일

| 기능 | 설명 |
| --- | --- |
| 설문 기반 닉네임 | 협업 성향을 반영한 한국어 닉네임으로 익명성과 개성을 함께 제공합니다. |
| 실시간 모둠 채팅 | Socket.IO 기반 모둠 대화와 채팅 히스토리를 제공합니다. |
| 전송 전 문장 조언 | 거친 표현이나 욕설을 보내기 전에 경고와 부드러운 수정안을 제안합니다. |
| 그룹 대화 개입 조언 | 대화 흐름을 주기적으로 분석해 참여 유도나 심화 질문을 추천합니다. |
| 역할 기반 기여 분석 | 단순 발화량이 아니라 질문, 요약, 자료 제안, 갈등 완화 등 역할 중심으로 분석합니다. |
